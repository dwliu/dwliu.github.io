初识ubuntu--我的Linux之旅

发表于 2007-07-21

最近几天比较闲，学着超频，把偶实验室的台机主频从1.8G超到2.25G，在everest的超频选项里看，从AMD Athlon 64 3000+变成了3500+，心里一阵窃喜，毕竟还没有加电压，直接在bios里面改外频，就超到这个效果，看来华擎这块主板超频真得不错！后来用找了几个小软件测了下系统，依然非常稳定（当然还要感谢实验室的中央空调）。看到网上有强人将其超到2.5G，小感慨一下，鉴于目前的超频幅度已经高达25%，也就不再费啥心思了。

把XP换成英文操作系统，也已经有几个月了。现在用起英文界面没有任何问题，而且越来越觉得还是英文看起来更舒服一些。电脑已经被我优化得飞快，像我这样一个对操作系统内核不感兴趣，又不是很喜欢编程的懒人，XP professional已经基本失去了再探索的价值。于是，前天晚上趁和sean一起吃饭，把他拖到实验室来，请他帮我装了一下Fedora Core 6。不得不声明一下，之前偶是一个对Linux一无所知的菜鸟，现在也是。

采用硬盘安装的方式，FC6的安装还是很快的，英文版的操作系统往往第一件事就是寻找中文支持。sean同学装完之后有事走了，然后偶就开始了辛苦的摸索。FC6刚装完无法识别NTFS分区，所以我硬盘上原来的东西都不能看。界面对中文的支持也很糟糕，所有的中文都是一个小方框，除了英文之外其他所有的字体都是乱码。唯一欣慰的是有Firefox可以上网，偶在linux下的第一件事，就在google.com里键入“fedora Chinese display”……

因为是菜鸟，网上的资料都看不懂，甚至连terminal都不知道怎么调出来-.-于是找到实验室的一个师兄，可惜师兄也很长时间没弄这个了，最后也还是不行，于是偶说师兄让我自己折腾一阵吧，又折腾了半个多小时后，未果。宿舍就要关门了，只好恨恨地离开了实验室。

回到宿舍，和阿q大体说了下，结果他说你干嘛用Fedora啊，什么界面不好云云，听得偶头都大了，然后昨天上午，他帮我把fedora换成了最新的ubuntu 7.10。坦白地说，ubuntu的界面很漂亮，终于不用看到fedora那个恶心的双螺旋DNA和一排排的小方框了，也很开心。安装后默认是桔红色的背景，很喜欢。而且面板上面可以添加许多方便的功能，如便笺、天气、网络、关机按钮等等，也可以新建不同的面板，比windows的任务栏更加灵活，并且集成了部分google desktop的功能。

现在的ubuntu 7.10，上网有Firefox，多媒体有MPlayer，办公有OpenOffice，聊天的话有pidgin和EVA（后面装的），除了还是英文的GUI。接下来的主要任务，就是从使用层面上，对中文输入法安装和设置的探索。
